## [0.1.0] - 2026-04-03

### Added
- Initial release of `animated-icons-vue`
- 402+ animated SVG icons for Vue 3
- Hover-triggered animations (translate, rotate, scale, opacity)
- Programmatic animation control via `startAnimation()` / `stopAnimation()`
- Props: `size`, `color`, `strokeWidth`, `active`, `className`
- Full TypeScript support
